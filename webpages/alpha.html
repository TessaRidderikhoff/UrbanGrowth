Alpha determines if there is randomness in the model. This means that a cell that does not have the highest potential can still be selected to change. In this way randomness comes into the model.

When alpha is for example 0 there is no randomness in the model. There are only large blocks with the same kind. When alpha grows, cells that have less potential to change also get a chance to change. So the higher alpha is, the more random spots appear in the landscape.

 

 

What is striking in the graph is that when alpha is very small (0 - 0.5), a large fractal dimension is immediately visible (~ 1.5). This dimension hardly grows when time steps are taken. If alpha is larger (1 or> 1), there is a very low fractal dimension (~ 0.7) at the start and this grows over time, the curve is logaritmic. This is understandable because when alpha gets bigger there is more randomness and this is not favorable for a large fractal growth.
