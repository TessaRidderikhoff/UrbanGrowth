The implemented model, which was developed by R. White and G. Engelen (1993), simulates urban land use over time using cellular automata. The specifications of the model are given below.

  • The model consists of a grid of cells, where the state of a cell represents a type of land use. The possible states of the cells are vacant (V), housing (H), industrial (I) and commercial (C).
  • At each iteration the growth rate determines the number of cells that will be converted to another state. The ratio in which cells are transformed to commerce, industry and housing is 1:4:7.
  • A cell's neighbourhood is the set of cells within a circle with a specified radius.
  • A hierarchal order is applied to the different states. This order is as follows: vacant (lowest state), housing, industry and commerce (highest state). A cell is allowed to change from a lower state to a higher state, but not the other way around.
  • At each iteration in the simulation, the transition potentials for all transitions allowed by the hierarchy are calculated using the following formula:
    Pi,j=S(1+ åh,k,dmk,dIh,d)
    where Pi,j is the transition potential from state i to state j;
    mk,d is the weighting parameter assigned to cells in state k in distance zone d;
    h is the index of cells within a given distance zone;
    Ih,d is equal to 1 if the state of cell h = k, otherwise it equals 0;
    S is a stochastic disturbance term and is given by:
    S=1+(-lnR)a  
    Here R is a random number between 0 and 1. The parameter α determines the size of the stochastic perturbation.
  • The transition potentials are used to determine which cells are transformed to other states. At each iteration, the cells having the highest transition potentials for a certain state will be converted to that state. However, if a certain cell belongs to the group of cells with the highest transition potentials for more than one state, then it is stochastically determined which transition takes place. In compensation, a cell having a lower transition potential will be converted to the other state, so that the correct number of transformations occur.
